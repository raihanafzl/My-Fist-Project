# THAT'S MY COIN! - Game PBO Pygame

## Deskripsi
Game sederhana berbasis Python menggunakan Pygame.
Player melawan bot untuk mengumpulkan koin sebanyak mungkin dalam waktu 30 detik.

## Fitur
- Player vs Bot
- AI sederhana (bot mengejar koin)
- Sistem skor
- Timer
- Background gambar
- Menu dan game over screen

## Cara Menjalankan
1. Install Python
2. Install pygame:
   pip install pygame
3. Jalankan program:
   python game.py
